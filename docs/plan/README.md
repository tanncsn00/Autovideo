## Pricing Chốt

```
FREE (GitHub)     → CLI, 5 providers, watermark
Monthly $7.99     → Desktop app, all features
Yearly $59.99     → Tiết kiệm 37%
Lifetime $89      → Dùng v1.x mãi mãi
Add-ons           → Voice Cloning $29, Team $99, Cloud $9.99/mo
Đại lý sỉ        → 35-50% off, 4 gói (20/100/300/1000 keys)
```

## Quick Numbers

- Revenue Năm 1 (1000 khách trực tiếp): ~$65,500
- Revenue Năm 1 (+ 2000 khách qua đại lý): ~$158,000
- Chi phí hạ tầng: ~$2/tháng (launch) → $70-140/tháng (growth)
- Marketing budget: ~$300-500/tháng (chủ yếu organic free)
- Build time: Desktop app 4-6 tuần + Business infra 3-4 tuần
